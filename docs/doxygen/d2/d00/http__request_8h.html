Detailed Description

Data structures and helpers for representing a parsed HTTP request.

This header defines fixed-size limits, the header and request structures, and lifecycle helpers to initialize and clear an http_request.

◆ http_request_clear()

void http_request_clear ( struct http_request * req)

Free all heap-allocated memory in an http_request struct.

Frees method, path, version, each header name/value, the headers array itself, and body if present. After this call, the structure is reset to an empty state (NULL pointers, zero counters).

Does not free the struct itself! If the req struct is heap allocated itself, it has to be freed by the owner.

Parameters
reqPointer to the http_request struct to free (must not be NULL).

◆ http_request_get_header_value()

const char * http_request_get_header_value ( const struct http_request * req,
const char * header_name )

Look up a header value by name (case-insensitive).

Scans req->headers and returns the value pointer of the first header whose name matches header_name, using a case-insensitive comparison.

Parameters
reqParsed request to search (must not be NULL).
header_nameHeader field-name to look up (must not be NULL).
Returns
  • Pointer to the header value (NUL-terminated) if found.
  • NULL if no matching header exists or if inputs are invalid.
Note
  • If multiple headers share the same name, the first occurrence is returned.
  • The returned pointer is owned by http_request and remains valid until http_request_clear() is called.

◆ http_request_init()

void http_request_init ( struct http_request * req)

Initialize an http_request struct to a safe empty state.

Sets all pointers to NULL and counters to zero so that http_request_clear() can be safely called later even if parsing fails midway.

Parameters
reqPointer to the request object to initialize (must not be NULL).
