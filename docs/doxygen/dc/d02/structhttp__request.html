Detailed Description

HTTP request struct for representation of a parsed HTTP request.

Fields method, path, version, and each header name/value are heap-allocated, null-terminated strings. They are freed by http_request_clear().

The headers array is also heap-allocated (size: num_headers). The body pointer is an optional heap-allocated, null-terminated copy of the message body (present only if a body was read/parsed).
