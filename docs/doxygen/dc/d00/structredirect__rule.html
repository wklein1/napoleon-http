Detailed Description

A single redirect rule.

Ownership & lifetime:

  • from and to point to externally owned, NUL-terminated strings (e.g., literals or configuration buffers). The registry does not free them.
  • from_len and to_len cache the byte lengths (excluding NUL) for faster lookup.

Semantics:

  • If append_tail is true and the rule matches via PREFIX/SEGMENT_PREFIX, the unmatched suffix of the request path ("tail") may be appended to to by the lookup implementation. EXACT rules must not use append_tail.
  • redirect_type is an app-level semantic (temporary/permanent; with/without method preservation).
