Detailed Description

Simple static-file router using the filesystem abstraction.

Matches requests by URL prefix and serves files from a given filesystem root. Only GET is handled. Results are returned as app_response with payload allocated on the heap (payload_owned = true).

Configuration and state for the static-file router.
