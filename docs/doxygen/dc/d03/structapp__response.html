Detailed Description

Application response to be serialized by the adapter.

Ownership & lifetime:

  • If payload points to dynamically allocated memory and payload_owned == true, the adapter/framework will free it after sending.
  • If payload points to static storage or memory owned elsewhere, set payload_owned == false (it will not be freed by the framework).
  • For APP_NO_CONTENT, set payload_len to 0 and leave payload as NULL.
