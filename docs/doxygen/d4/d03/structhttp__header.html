Detailed Description

One header line (name: value).

Ownership:

  • If name_owned is true, the framework/parser will free(name).
  • If value_owned is true, the framework/parser will free(value).
  • If false, the pointers must refer to static/foreign storage and will not be freed.

Note: This struct is used by both request and response code paths. Strings are NUL-terminated. Ownership is indicated per field via *_owned flags. If a fieldâ€™s *_owned flag is true, the respective pointer must be heap-allocated and will be freed by the corresponding clear() routine.
