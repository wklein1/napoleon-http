Detailed Description

Virtual filesystem handle shared by the server.

  • root is a constant docroot/mount prefix and must outlive this struct.
  • ctx holds implementation-defined state (mount handle, partition pointer, etc.).
  • ops points to the function table supplied by the implementation.
